Práctica 2: Hola Mundo (LED integrado) 25 min
ProgramaciónSalidas digitales
Cargando..., vas a hacer parpadear el LED de la tarjeta (pin 2). Crea un nuevo sketch, escribe esto y carga:
// los // indican que lo que esta adelante son comentarios y no codigo // Solo se ejecutara una sola ves cuando recibe corriente el ep32 void setup() { pinMode(2, OUTPUT); // se configura el puerto como un puerto de salidad } // una ves que ejecuta setup, se quedara dentro de loop repitiendo el ciclo cada vez. void loop() { //void -> vacio | loop bucle infinito ()-sin parametros { corchete de apertura para abrazar el contenido de la funcion loop digitalWrite(2, HIGH); // se eleva el interrutor del puerto para darle una salida de voltaje de 3.3v delay(500); // se hace una pausa general de 1/2 segundo digitalWrite(2, LOW); // se baja el interruptor del puerto para dejar de pasar voltaje delay(500); // se hace una pausa general de 1/2 segundo } // } corchete de cierre indica el cierre del contenido de la funcion loop
✅ Si parpadea, ¡ya lo lograste! Ese es tu primer programa.
❓ ¿Por qué no parpadea? Revisa que seleccionaste bien el puerto y la placa.

❓ Preguntas que se hizo Cargando... (y sus respuestas)

¿Por qué mi LED no enciende?
Revisa polaridad (ánodo largo al pin, corto a GND), la resistencia, que el pin sea el correcto y esté en OUTPUT.
No encuentro el puerto COM del ESP32
Instala el driver CP210x o CH340 según tu modelo. Reinicia Arduino IDE.
¿Puedo dañar el ESP32?
Si no conectas 5V en pines equivocados, es difícil. Respeta voltajes (3.3V en pines).
Mi código compila pero no parpadea
Verifica que el pin del LED sea 2 (o el que usaste). Algunas placas usan LED en GPIO2, otras en GPIO13, checa tu modelo.
