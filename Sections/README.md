# ZonaMarkBot

Un bot modular en Python que se ejecuta localmente con Flask y se integra en una página web servida desde XAMPP (https://zonamark/ZM/).

## Estructura
Sections/
├── data/           # Configuración (sinónimos)
├── src/            # Código fuente (Flask + bot)
├── templates/      # HTML
├── static/         # CSS y JS
├── requirements.txt
└── README.md
